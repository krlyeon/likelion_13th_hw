4/2 Django session 강의 요약

개발 세팅
  1. 개발 환경 세팅
    • pipenv shell을 통한 가상 환경 실행
  2. gitignore 파일 준비
    • db, media와 같은 충돌 위험 파일을 깃 추적에서 제외
  3. Templates 디렉토리 생성
    • HTML, CSS 등 사용자에게 보여줄 레이아웃과 구조를 한 곳에 정리
  4. views.py 작성
    • 작성한 HTML 파일을 연결하기 위한 view 함수 구성
  5. urls.py 작성
    • 원하는 URL 경로에 해당하는 view 연결 설정
  6. 앱 실행
    • python manage.py runserver 명령어로 로컬 서버 실행

링크 연결

  • 형식: {% templatetag openblock %} url 'intropage' {% templatetag closeblock %}
  • urls.py에서 지정한 name을 기반으로 페이지 간 이동 구현

템플릿 변수 사용법

  • 변수명: 기본 변수 출력
  • 변수명.속성명: 객체 속성 접근
  • 리스트.인덱스: 리스트의 특정 요소 접근
  • 딕셔너리.key: 딕셔너리 값 접근
  • 템플릿 내 for, if 등의 문법 사용 가능

템플릿 상속

  • {% templatetag openblock %} block content {% templatetag closeblock %}: 다른 템플릿에서 채워질 부분 정의
  • {% templatetag openblock %} endblock {% templatetag closeblock %}: 블록 종료
